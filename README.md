# Agent Skills

Stable, reusable agent skills maintained by Game Dev RTA Club.

## Skills

- `rubber-duck-caller`: opens and continues an independent review dialogue.
- `rubber-duck-partner`: reviews a draft, decision, or result inside that dialogue.

Related skills share a name prefix. Every skill still lives directly under `skills/`.

## Install

Install every skill into the current project:

```bash
gh linked-skills install game-dev-rta-club/agent-skills --all
```

The repository also includes Codex and Claude plugin metadata.
